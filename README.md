# 🎯 Advice Generator UI

A simple and clean web app that generates random advice for users with a single click. Built with a focus on minimal design, responsiveness, and smooth user interaction.

## 📦 Installation & Setup


Navigate into the project folder:

```bash
cd advice-generator-ui
```

Install dependencies (if applicable):

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 🔌 API Used

This project uses the Advice Slip API to fetch random advice:

👉 https://api.adviceslip.com/
